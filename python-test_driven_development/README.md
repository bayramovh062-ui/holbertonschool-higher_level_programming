# Python - Test Driven Development

This project is about learning Test-Driven Development (TDD) in Python.
The main objective is to learn how to write `doctest` and `unittest` for Python code, ensuring that all edge cases are covered before actually writing the implementation.

## Project Structure
* `tests/` - Directory containing all text files for `doctest`.
* `0-add_integer.py` - Function that adds two integers.
* `tests/0-add_integer.txt` - Test cases for `0-add_integer.py`.
